Welcome to the Resource Oriented Template Language project.

The Resource Oriented Template Language (rstl pronounced \'rest-É™l\ ) allows web developers to build web sites using a simple template language syntax inspired by Django templates. The template language features template and site inheritance and allows the layout and structure of pages in the web application to be cleanly separated from the resources that make up a page. The template engine has bindings for Java, which allows developers to render these templates in any Java based application or application server.
